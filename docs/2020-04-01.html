2020-04-01

  • JavaScript和ECMAScript的完整功能列表：JavaScript|MDN

  • 推荐macOS上一款很好用的截图工具：Xnip

    编辑截图和滚动截图功能很喜欢，可惜滚动截图上有水印，去除水印需要购买pro，贫穷让我接受了这个缺点。

  • 利用npm全局安装Babel遇到了一个bug：EACCES current user ("nobody") does not have permission to access the dev dir "/Users/shawvey/Library/Caches/node-gyp/12.16.1"

    解决方案: NPM全局安装依赖权限报错Error: EACCES: permission denied

  • 利用Babel转译js文件，遇到了ReferenceError: Unknown plugin "add-module-exports" specified in "folder/.babelrc" at 0, attempted to resolve relative to "folder":

    解决方案：Module build failed: ReferenceError: Unknown plugin "add-module-exports" #120

    npm install babel-plugin-add-module-exports --save-dev
npm install babel-preset-es2015 --save-dev

    但此时又遇见了另外一个问题：

    查看上面指令运行时的提示发现：npm install babel-preset-es2015 --save-dev不再推荐使用

    按推荐改为使用npm install babel-preset-env，运行成功！

  • 有关 Node.js 支持原生 ES2015 导入功能的信息可以在 Node.js支持原生ES2015导入功能 查阅。
